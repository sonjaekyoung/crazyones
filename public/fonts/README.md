# Poster Fonts

- `Vertiky.ttf` is used to match the bounty lettering approach in
  `YuskaWu/one-piece-wanted-poster`.
- `Griun_PolSensibility-Rg.ttf` is the project's existing Korean display font,
  copied here so canvas rendering can address it by a stable public URL.

The poster renderer does not bundle the original One Piece poster image assets.
